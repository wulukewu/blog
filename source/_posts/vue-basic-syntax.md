# Vue Syntax

`v-on`: `@click`
`v-bind`: `:input`, `:input.enter`, `:key`
`v-model`:
`v-if`, `v-else-if`, `v-else`
`v-show`
`v-for`: `v-for="goal in goals"`

`props`: parent to child
`emits`: child to parent
